### 0.1.0

First release. Ridge support for JetBrains IDEs via `ridge-lsp` and LSP4IJ:
syntax highlighting (the TextMate grammar shared with the VS Code extension),
diagnostics, hover, go-to-definition and type-definition, find usages, rename,
signature help, formatting, completion, and call/type hierarchy. Resolves the
server from `~/.cargo/bin` or `PATH`, with an optional path override under
**Settings | Languages & Frameworks | Ridge**.
